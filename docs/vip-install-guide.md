## ⚠️ Что должно быть ДО установки

1. **OpenClaw с первой недели курса** — если ещё не ставили, сначала первый
   установщик. Команда из материалов недели 1.
2. **Оплачен VIP-тариф** — если ещё нет, пишите в поддержку.
3. **5 Telegram-ботов** созданы через `@BotFather` (по одному на каждого агента).
   Названия любые: «Мой Технарь», «Мой Маркетолог» и т.д. Сохраните токены.
4. **Ваш Telegram user ID** — узнать через `@userinfobot` (напишите ему что угодно,
   он пришлёт ваш ID).

Если хоть одного пункта нет — сделайте сначала. Без них установщик не запустится.

---

## Шаг 1. Получите VIP-токен (1 минута)

1. Откройте в Telegram бота **[@AITeamVIPBot](https://t.me/AITeamVIPBot)**
2. Напишите `/start`
3. Введите **email** или **телефон**, которыми вы оплачивали VIP-тариф
4. Бот пришлёт токен вида:
   ```
   VIP-A7D2E8C09F1B4A3D-975494053-eyJhbGciOi...<длинная строка>
   ```
5. Скопируйте его — нужен на шаге 3.

### 🔒 Критически важно

> Пишите боту `@AITeamVIPBot` с того же Telegram-аккаунта, с которым будете
> писать вашим AI-агентам. **Токен привязан к вашему Telegram-аккаунту** —
> это защита от шаринга. Если получите токен с одного TG, а ставить будете
> с другого — установщик откажет.

---

## Шаг 2. Запустите установщик (1 минута)

Откройте терминал:

- **На Mac**: `Cmd + Space` → напечатайте `Терминал` → Enter
- **На Windows**: `Win` → напечатайте `powershell` → Enter
- **На VPS**: подключитесь по SSH к вашему серверу

Скопируйте команду и вставьте:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-agents-pack/main/scripts/install-agents.sh)
```

Нажмите Enter. Установщик запустится и покажет меню.

---

## Шаг 3. В меню выберите **«2) VIP — 5 агентов»** (5-8 минут)

Установщик проведёт вас по шагам, нужно будет:

1. **Вставить VIP-токен** (шаг V1) — тот, что прислал `@AITeamVIPBot`
2. **Выбрать модель AI** (шаг R1) — нажмите Enter для рекомендации
3. **Ввести 5 Telegram bot tokens** (шаг R2) — по очереди:
   - Технаря
   - Маркетолога
   - Продюсера
   - Дизайнера
   - Координатора
4. **Подтвердить ваш Telegram ID** (шаг R2.5) — обычно берётся автоматически

### 💡 Совет

**Заранее запишите** где-то под рукой, какой токен от какого бота — скрипт
будет спрашивать их по порядку, и если перепутаете (токен от Маркетолога
для Технаря) — все сообщения уйдут не к тому агенту.

## Шаг 4. Проверка (1 минута)

Когда установщик закончит — откройте Telegram, найдите любого из ваших 5
ботов (установщик напишет их `@username` в финале) и напишите `/status`.

Должен ответить в течение 5-10 секунд. Если ответил — **всё работает!** 🎉

Напишите ему обычное сообщение — получите ответ AI.

## 🆘 Если что-то пошло не так

### «Токен не прошёл проверку»

Скопируйте токен ещё раз целиком из `@AITeamVIPBot`. Проверьте что в нём
**три секции через дефис** и последняя (подпись) не обрезана. Формат:
`VIP-XXXX...XXXX-ЦИФРЫ-длинная_строка`.

### «Этот токен выдан для другого TG»

Вы пишете боту `@AITeamVIPBot` с одного Telegram-аккаунта, а устанавливаете
с другого. Откройте `@AITeamVIPBot` с **того же TG**, который используете
для AI-агентов, получите новый токен.

### «Не нашёл настройки OpenClaw»

Сначала нужен первый установщик (материалы недели 1). Запустите его,
подключите один базовый Telegram-канал — и вернитесь сюда.

### «declare: -A: invalid option» или странная ошибка bash

Старая версия bash на macOS (3.2). Установщик должен сам поставить
новую через Homebrew. Если не получилось — выполните:
```bash
brew install bash
```
и запустите установщик снова.

### Ничего из этого не помогло

Соберите debug-bundle и пришлите в саппорт:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-agents-pack/main/scripts/install-agents.sh) --collect-debug
```

Zip-файл появится в `~/openclaw-agents-pack-debug-*.zip`. Все секреты в
нём замаскированы — можно смело пересылать.

## ❓ Частые вопросы

**Что если я хочу сменить Telegram-аккаунт?**
Напишите в саппорт. Мы обновим привязку токена к новому TG ID, и
старый токен перестанет работать.

**Могу ли я установить VIP на два компьютера (ноут + VPS)?**
Да. На обоих используйте один и тот же VIP-токен и тот же Telegram-аккаунт.
Каждая машина будет независимой инсталляцией.

**Что если у меня уже стоят 3 агента (Standard)?**
Запустите установщик, выберите «VIP — 5 агентов», на шаге «перезаписать
начисто?» нажмите Enter — старые 3 будут пересозданы + добавятся 2 новых
(Дизайнер + Координатор).

**А если я хочу вернуться на Standard?**
Просто запустите установщик, выберите «Стандарт — 3 агента». Но зачем? 🙂
