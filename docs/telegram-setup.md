# 🤖 Как создать три Telegram-бота через @BotFather

**Для кого**: ученики, которые запускают `install-agents.sh` и упёрлись в «Введите токен».

**Что получите**: три отдельных токена — по одному для Технаря, Маркетолога, Продюсера.

---

## Зачем три разных бота

OpenClaw роутит Telegram-сообщения по **`accountId`**: каждый бот = отдельный
«аккаунт» в конфиге → отдельный агент. Один токен не может обслуживать троих
агентов: кто бы ни написал — ответит только первый попавшийся бинд.

Поэтому: три бота через @BotFather, три разных токена. Занимает 5 минут.

---

## Шаг 1 — открыть @BotFather

В Telegram найдите **`@BotFather`** (он с синей галочкой). Нажмите «Старт»
если не начинали раньше.

---

## Шаг 2 — создать бота Технаря

1. Отправьте `/newbot`
2. BotFather спросит имя — что угодно, но так, чтобы вы сами помнили:
   - Например: `Мой Технарь` или `AntonsTechBot`
3. Теперь username (должен заканчиваться на `bot`):
   - Например: `ivan_tech_bot`, `mycompany_tech_bot`
   - Если занят — добавьте цифру
4. BotFather пришлёт сообщение с **токеном** вида:
   ```
   7123456789:AAGkABCDEFghijklmnopqrstuvwxyz123456789
   ```
5. **Скопируйте его** — вставите в установщик.

---

## Шаг 3 — то же самое для Маркетолога

Повторите `/newbot` в том же чате с @BotFather. Имя/username — на ваш вкус,
например `Мой Маркетолог` → `ivan_marketer_bot`.

Скопируйте **новый** токен (он будет другим).

---

## Шаг 4 — и для Продюсера

Снова `/newbot`, например `Мой Продюсер` → `ivan_producer_bot`.

Скопируйте третий токен.

---

## Итого — у вас три токена

```
Технарь:    7123456789:AAGkAAAABBBBCCCC...
Маркетолог: 7123456790:AAGkXXXXYYYYZZZZ...
Продюсер:   7123456791:AAGk1111222233334...
```

Храните их временно в заметках — пока установщик спросит. **После установки
удалите их из заметок** (они лежат в `~/.openclaw/openclaw.json` с правами 600,
там безопаснее).

---

## Опциональные настройки в @BotFather (можно пропустить)

Чтобы боты выглядели профессиональнее в Telegram — можно через `/setavatar`,
`/setdescription`, `/setabouttext`. Но это **не нужно** для работы.

Пример описания для Технаря:
> Персональный технический ассистент. Помогаю с терминалом, API, деплоями.
> Отвечаю коротко и по делу. Работаю на OpenClaw.

Пример описания для Маркетолога:
> Мой маркетолог. Помогаю с оффером, воронкой, трафиком. Даю одну лучшую
> рекомендацию, не «5 вариантов». Работаю на OpenClaw.

---

## Что делать, если потерял токен

BotFather: `/mybots` → выберите бота → **API Token**. Он покажет токен снова
(это не «потеряно навсегда»). Но если боитесь утечки — **Revoke current token**,
и BotFather сгенерирует новый.

После revoke старый токен перестанет работать — нужно будет обновить через
`openclaw-factory-reauth` или переустановить агентов.

---

## Безопасность

- **Не отправляйте токены в публичные чаты.** С токеном можно читать всю
  переписку бота и отвечать за него.
- **Если токен утёк** (засветили в скриншоте, публичном коде) — идите в
  @BotFather → `/mybots` → ваш бот → **Revoke current token**.
- **allowlist**: установщик спросит ваш Telegram user ID и пропишет бота так,
  чтобы он отвечал только вам. Без этого — бот ответит любому, кто напишет.

Всё, к установщику.
