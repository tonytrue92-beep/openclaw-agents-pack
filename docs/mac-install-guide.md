# 🍎 Установка на macOS — DMG (двойной клик)

> ⏱ **Время:** 15-30 минут (включая создание ботов в @BotFather)
> 💻 **Поддерживается:** macOS 10.15 (Catalina) и новее

Этот гайд — для тех кто хочет поставить **без терминала и копи-пасты команд**. Скачиваешь `.dmg` файл, открываешь его, делаешь двойной клик на 2 файла по очереди — готово.

> 📘 Если предпочитаешь команды в терминале — см. [`vip-install-guide.md`](./vip-install-guide.md). Делается то же самое, путь немного длиннее.

---

## 🚀 Полный путь — 5 шагов

### Шаг 1 — Скачай DMG

Открой страницу релизов: https://github.com/tonytrue92-beep/openclaw-agents-pack/releases/latest

Скачай файл **`OpenClaw-Setup.dmg`** (~100-200 KB).

`[SCREENSHOT: страница releases с подсвеченным OpenClaw-Setup.dmg]`

### Шаг 2 — Открой DMG

Двойной клик по скачанному `.dmg` файлу. Откроется окно Finder с тремя файлами:

```
📄 1. Установить OpenClaw.command
📄 2. Установить AI-команду.command
📄 README.txt
```

`[SCREENSHOT: окно DMG с тремя файлами]`

### Шаг 3 — Подготовь то что понадобится

**Перед запуском** возьми вот это (займёт 5-10 минут):

| Что | Где взять |
|---|---|
| **Курс-токен** | Telegram → [@AITeamVIPBot](https://t.me/AITeamVIPBot) → `/start` → введи email/телефон оплаты курса. Получишь `STD-...` (Standard) или `VIP-...` (VIP). |
| **Telegram-боты** | [@BotFather](https://t.me/BotFather) → `/newbot` → создай 3 ботов (Standard) или 6 (VIP). Сохрани каждый токен. |
| **Свой TG user ID** | [@userinfobot](https://t.me/userinfobot) → `/start` → бот пришлёт твой ID (число типа `123456789`). |
| **(Опционально) OpenAI ключ** | Только если хочешь embedding-память. См. [`docs/openai-key-setup.md`](./openai-key-setup.md). Карта РФ напрямую не работает. |

### Шаг 4 — Запусти первый файл

Двойной клик на **`1. Установить OpenClaw.command`**.

#### ⚠️ Если macOS показал предупреждение

При **первом** запуске покажет:
> «Файл нельзя открыть, потому что разработчик не подтверждён»

Это **нормально**. Что делать **один раз**:

1. Закрой окно с предупреждением (`Cancel` / `Отмена`)
2. **Правый клик** (или Control+клик) на файле `.command`
3. В меню выбери **«Открыть»** (Open)
4. В новом окне нажми **«Открыть»** ещё раз
5. Дальше работает без вопросов. Повтори то же для второго файла.

`[SCREENSHOT: Gatekeeper-предупреждение + правый клик → Open]`

После открытия — запустится **Terminal**, покажет инструкцию, попросит нажать Enter, потом запустит установщик OpenClaw. Установщик сам спросит:
- **Курс-токен** (вставь полученный в шаге 3)
- **API-ключ модели** (найди в материалах курса)
- **Токен бота для main-агента** (один из созданных в @BotFather)

⏱ Дождись пока бот напишет тебе в Telegram — ~5-10 минут.

### Шаг 5 — Запусти второй файл

Когда бот первого установщика ответит в Telegram — **закрой окно Terminal от шага 4** (или оставь открытым) и сделай двойной клик на **`2. Установить AI-команду.command`**.

> При первом запуске опять покажет Gatekeeper-warning — обойди тем же способом (правый клик → Открыть).

В меню выбери:
- **`1) Стандарт`** — если у тебя тариф Standard (3 агента)
- **`2) VIP`** — если VIP (6 агентов)

Установщик запросит токены остальных ботов по порядку (Технарь → Маркетолог → Продюсер → ... ). **Курс-токен повторно НЕ просит** — он уже сохранён с шага 4.

⏱ ~5-10 минут — все агенты будут установлены.

### Шаг 6 — Проверка

Открой Telegram, найди созданных ботов, напиши **`/status`** каждому. Должны ответить в течение 5-10 секунд.

У VIP-агентов (Дизайнер, Координатор, Копирайтер) при первой встрече начнётся онбординг — они зададут 5-6 вопросов про твой бизнес. Ответь — это разово, дальше будут работать с этим контекстом.

## 🆘 Если что-то пошло не так

### Двойной клик ничего не делает

`.command` файлу нужен executable bit. Проверь в Terminal:
```bash
ls -la /Volumes/OpenClaw\ Setup/*.command
```
Должно показать `-rwxr-xr-x` (3 буквы `x`). Если нет — DMG битый, скачай заново.

### macOS показывает «приложение повреждено» (на M1/M2/M3)

Это редкий quarantine-баг на Apple Silicon. Очисти карантин-флаг:
```bash
xattr -dr com.apple.quarantine /Volumes/OpenClaw\ Setup/
```

### `bash <(curl)` падает с timeout

Скорее всего интернет режется (VPN / корп. фаервол / медленный VPS). Попробуй:
1. Отключить VPN на время установки
2. Или использовать командный путь напрямую — см. [`vip-install-guide.md`](./vip-install-guide.md)

### Установщик зависает на «индексирую память»

Embedding-индексация может занять минуту на больших MEMORY.md. Если больше 2 минут — Ctrl+C, потом запусти диагностику в Terminal:
```bash
bash <(curl -fsSL https://github.com/tonytrue92-beep/openclaw-agents-pack/releases/latest/download/install-agents-bundled.sh) --diagnose-only
```

### Бот молчит после `/start`

Не запускай переустановку. Запусти `--diagnose-only` (см. выше) — покажет что именно сломано (gateway / Telegram / auth-profile).

### Какой-то другой странный сбой

Собери debug-bundle для саппорта:
```bash
bash <(curl -fsSL https://github.com/tonytrue92-beep/openclaw-agents-pack/releases/latest/download/install-agents-bundled.sh) --collect-debug
```

Файл `~/openclaw-agents-pack-debug-*.zip` появится в Home. Все секреты в нём замаскированы — пришли в саппорт-чат курса.

## ❓ FAQ

**Q: Зачем 2 файла, нельзя одним?**
A: OpenClaw (движок) и AI-команда (агенты) — разные продукты. Один ставит платформу, второй — команду на ней. В реальной жизни иногда хочется сначала просто OpenClaw для отладки, а уж потом агентов — поэтому разделено.

**Q: Можно сразу запустить второй файл?**
A: Нет. Второй файл проверит что OpenClaw уже стоит и откажется работать иначе. Это защита — без работающего OpenClaw агентам некуда «приземлиться».

**Q: Зачем Gatekeeper-предупреждение, если в коде ничего опасного?**
A: macOS требует чтобы разработчик платил Apple $99/год за официальную подпись (Apple Developer Program). Мы маленький проект, не платим — но все скрипты внутри открытые, можешь посмотреть в любом текстовом редакторе. После «правый клик → Открыть» один раз — больше не спросит.

**Q: Как обновить агентов после установки?**
A: Не нужно ставить заново. В Terminal:
```bash
bash <(curl -fsSL https://github.com/tonytrue92-beep/openclaw-agents-pack/releases/latest/download/install-agents-bundled.sh) --refresh-templates
```
Это обновит характеры агентов, но **не** тронет память (`MEMORY.md`) и ответы онбординга (`USER.md`).

**Q: Можно поставить на Mac и на VPS одновременно?**
A: Да. Используй один и тот же курс-токен и один и тот же Telegram-аккаунт для агентов. Каждая машина — независимая инсталляция. Память агентов не синхронизируется (это отдельная фича на будущее).

**Q: Не получается из России — карта в OpenAI не проходит?**
A: Это санкции, обхода нет. Решение — виртуальная зарубежная карта за 10 минут. См. [`docs/openai-key-setup.md`](./openai-key-setup.md), раздел про РФ-карты.

**Q: Я вообще не понимаю что такое Telegram-бот, как создать?**
A: Открой в Telegram → найди `@BotFather` → нажми Start → команда `/newbot` → следуй инструкциям мастера. Подробно с скриншотами — в [`vip-install-guide.md`](./vip-install-guide.md), Шаг 1.
