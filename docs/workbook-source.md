# 📘 Workbook source — «Установка и обновление AI-команды»

> **Продюсеру:** это исходник для workbook'а. Ниже — всё что нужно
> передать ученику: термины, команды, скриншоты (места отмечены
> `[SCREENSHOT: ...]`), проверочные чек-листы и практические задания.
>
> Структура готова под вёрстку — 5 модулей + 3 приложения. Прикинул
> ориентировочное время на каждый модуль.
>
> Что **обязательно** добавить при вёрстке:
> - Скриншоты в местах `[SCREENSHOT: описание]`
> - Брендовое оформление курса
> - Поля для записи ответов в практических заданиях
> - Нумерацию страниц

---

## 🎯 Для кого этот workbook

Для учеников курса которые:
- Купили доступ к AI-команде (Standard или VIP тариф)
- Хотят **самостоятельно** установить агентов на свой компьютер
  за 10–15 минут без знания командной строки
- Потом будут обновлять их по мере выхода новых версий

**Что получит в итоге:**
- 3 агента (Standard) или 6 агентов (VIP) в Telegram, каждый в своём боте
- Память, которая накапливается от сессии к сессии
- Расширяющиеся умения (для VIP — онбординг-протокол, skills, characters)

**Ориентировочное время прохождения:** 60–90 минут первая установка,
2 минуты на каждое последующее обновление.

## 📚 Глоссарий (на 1-ю страницу workbook'а)

| Термин | Что значит простыми словами |
|---|---|
| **OpenClaw** | Движок который связывает Telegram с ИИ-моделью. Стоит локально, работает 24/7. |
| **Агент** | Один AI-сотрудник со своей ролью, характером и памятью. Сидит в своём Telegram-боте. |
| **Telegram-бот** | Телеграм-аккаунт-робот. Создаётся бесплатно через `@BotFather`. На каждого агента — один бот. |
| **VIP-токен** | Ключ доступа к VIP-тарифу курса. Получается в `@AITeamVIPBot` по email/телефону оплаты. Привязан к TG-аккаунту — не передать другому. |
| **BotFather** | Официальный Telegram-бот для создания ботов. Через него получаются API-токены ботов. |
| **Workspace агента** | Папка на компьютере где лежит характер агента, его память, инструкции. `~/.openclaw/workspace-<имя>/` |
| **MEMORY.md** | Файл памяти агента. Здесь накапливается всё что агент узнал про тебя и задачи. **Ценность — в этом файле.** |
| **USER.md** | Что агент знает про тебя: ниша, ЦА, тон, запреты. Заполняется через онбординг. |
| **Онбординг** | Короткое знакомство — агент задаёт 5–6 вопросов про твой бизнес при первой встрече. |
| **Refresh** | Обновление только шаблонов без потери MEMORY.md и USER.md. |
| **SKILL.md** | Описание конкретного умения агента (например, reef-copywriting с 6 фреймворками). Лежит в `skills/`. |
| **Терминал** | Приложение для ввода команд. На Mac: Cmd+Space → «Терминал». На Windows: PowerShell. |

# 📗 МОДУЛЬ 1: Установка OpenClaw (движок)

⏱ **15–25 минут** · 🎯 **Цель:** поставить основу на которой будут жить агенты

## Что понадобится заранее

- [ ] Компьютер с macOS, Linux или Windows
- [ ] Стабильный интернет (скачается ~200 МБ)
- [ ] **10 ГБ свободного места** на диске
- [ ] Админ-права (на Mac/Linux — пароль пользователя, на Windows — подтверждение UAC)

## Шаг 1.1 — Открой терминал

**На Mac:**
1. Нажми `Cmd + Space` — откроется Spotlight
2. Напечатай «Терминал» и нажми Enter

`[SCREENSHOT: Spotlight с набранным словом "Терминал", стрелка на приложение]`

**На Windows:**
1. Нажми клавишу `Win`
2. Напечатай «powershell»
3. Нажми Enter

`[SCREENSHOT: Windows search с powershell]`

**На Linux:**
Знаешь как открыть терминал — открывай. Если нет — `Ctrl+Alt+T`.

> 💡 **Если боязно видеть «чёрное окно с текстом»** — это нормальное
> состояние терминала. Ничего не может сломаться если не вводить
> случайные команды.

## Шаг 1.2 — Скопируй и вставь команду установки OpenClaw


**Действия:**
1. Выдели команду выше (1 строка)
2. Скопируй (`Cmd+C` / `Ctrl+C`)
3. Переключись на терминал
4. Вставь (`Cmd+V` / в Windows PowerShell — правый клик мыши)
5. Нажми Enter

⏱ **Процесс займёт 5–10 минут.** Можно параллельно готовить следующие шаги (Модуль 2).

1. Открой [@BotFather](https://t.me/BotFather) в Telegram

## Шаг 2.3 — (для embedding-памяти) Получи OpenAI API-ключ

> «Без embedding» — этот ключ не нужен. Но рекомендуется (агенты
> работают быстрее и дешевле через 2-3 месяца).

**Краткая суть:** OpenAI берёт $0.13 за 1 миллион токенов индексации.
Активный клиент тратит **$0.20-1/месяц**. $5 депозит хватит на годы.

**Шаги:**

1. Открой [https://platform.openai.com/signup](https://platform.openai.com/signup)
2. Зарегистрируйся (или войди если уже есть ChatGPT-аккаунт — тот же
   email/пароль)

4. Нажми синюю кнопку **`+ Create new secret key`**
5. Назови `openclaw-embedding`, нажми `Create secret key`
6. **Скопируй ключ** `sk-proj-…` (показывается ОДИН РАЗ — потом не увидишь)
7. Перейди на [billing](https://platform.openai.com/settings/organization/billing)
8. Добавь карту → `Add to credit balance` → **минимум $5**

# 📗 МОДУЛЬ 4: Первый контакт — онбординг VIP-агентов

⏱ **10–15 минут** · 🎯 **Цель:** дать трём VIP-агентам узнать твой
бизнес — иначе они будут отвечать «обобщённо про эксперта»

> ⚠️ **Этот модуль — только для VIP.** Standard-агенты (tech/marketer/producer)
> работают без онбординга, но и глубины у них меньше.

## Зачем онбординг

Дизайнер, Координатор и Копирайтер при первом сообщении **сами
спросят 5–6 коротких вопросов** про:
- Чем занимаешься / что продаёшь
- Кто твоя ЦА
- Какой тон нужен
- Какие есть запреты / стоп-слова

Ответы записываются в `USER.md` агента — **один раз**. Дальше они
работают с этим контекстом и не пишут «обобщённо».

## Шаг 4.1 — Начни с Копирайтера (он пишет тексты — сразу увидишь разницу)

1. Открой бота Копирайтера в Telegram
2. Напиши `/start` или просто «привет»

### ✅ Обновится:

- `MEMORY.md` — **вся твоя накопленная память** (контекст сессий, сработавшие заголовки, стоп-слова)
- `USER.md` — **твои ответы онбординга** (ниша, ЦА, тон)

# 📎 ПРИЛОЖЕНИЕ B: Типичные проблемы и решения

## «command not found: openclaw»

**Причина:** OpenClaw не установлен или не прописан в PATH.

**Решение:**
```bash
# 1. Перезапусти терминал (закрой окно, открой новое)
# 2. Если не помогло — переустанови OpenClaw:
bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-factory/main/scripts/demo-install.sh)
```

## «Этот токен выдан для другого TG»

**Причина:** VIP-токен получен с одного Telegram-аккаунта, а
устанавливаешь с другого.

**Решение:**
1. Открой `@AITeamVIPBot` с того же TG что используешь для агентов
2. Получи **новый** VIP-токен (старый больше не работает)
3. Запусти установщик заново

## «declare: -A: invalid option» или странная bash-ошибка

**Причина:** устаревший bash на macOS (3.2).

**Решение:**
```bash
brew install bash
# Запусти установщик снова — он автоматически использует новый bash
```

Если нет Homebrew — ставь с [brew.sh](https://brew.sh).

## Бот не отвечает в Telegram

**Диагностика по пунктам:**

1. **Подожди 30 секунд** — первый запрос может быть дольше.

2. **Проверь что агент жив:**
   ```bash
   openclaw agents list
   ```
   Должен быть в списке.

3. **Проверь маршруты (routing):**
   ```bash
   openclaw agents bindings
   ```
   Должна быть привязка «бот ↔ агент».

4. **Собери debug-bundle и отправь в саппорт:**
   ```bash
   bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-agents-pack/main/scripts/install-agents.sh) --collect-debug
   ```
   Файл появится в `~/openclaw-agents-pack-debug-*.zip`. **Все секреты
   замаскированы** — можно смело пересылать в чат курса.

## Установщик «подвисает» на этапе Homebrew

**Причина:** медленное интернет-соединение + большой apt/brew-индекс.

**Решение:** подожди 5–10 минут. Если прошло больше 15 — прерви
(Ctrl+C), перезапусти установщик. Частично установленные куски
будут пропущены на втором запуске.

## После обновления агент пишет «обобщённо», не помнит контекст

**Маловероятно** — `--refresh-templates` не трогает MEMORY.md.
Но если реально произошло:

```bash
# Откати к последнему бэкапу
ls ~/.openclaw/workspace-<агент>/.backups/
# Скопируй последний бэкап
cp ~/.openclaw/workspace-<агент>/.backups/<самая-последняя-папка>/* \
   ~/.openclaw/workspace-<агент>/
```

Если такое произошло — обязательно напиши Антону в чат курса с
debug-bundle'ом.

## Потерял токен VIP-бота

Напиши Антону в саппорт-чат курса. Нужно будет **обновить привязку
токена к твоему TG ID**.

# 📎 ПРИЛОЖЕНИЕ C: FAQ (частые вопросы)

**Q: Могу ли я установить на два компьютера (ноут + VPS)?**
A: Да. На обоих используй один и тот же VIP-токен и один и тот же
TG-аккаунт. Каждая машина — независимая установка со своей MEMORY.

**Q: Что будет если я сменю Telegram-аккаунт?**
A: Старые токены ботов продолжат работать, но VIP-токен — нет
(он привязан к TG). Напиши в саппорт — обновим привязку.

**Q: Могу ли я перенести MEMORY.md на другой компьютер?**
A: Да. Скопируй папки `~/.openclaw/workspace-*/` на новую машину
после установки там OpenClaw + агентов.

**Q: Что если OpenClaw выпустит обновление?**
A: Запусти первую команду (установка OpenClaw) снова — обновится
до последней версии. Агенты и их память не пострадают.

**Q: А если я хочу сменить Standard → VIP?**
A: Запусти установщик, выбери «VIP — 6 агентов». Увидишь
R0-меню «обнаружен апгрейд» → выбери «Дополнить» → поставятся
только недостающие 3 агента. Старые 3 не трогаются.

**Q: Где хранится вся моя переписка с агентами?**
A: В `~/.openclaw/memory/` (короткий контекст) и
`~/.openclaw/workspace-<агент>/MEMORY.md` (долгосрочная память).

**Q: Можно ли настроить агента отвечать не только мне?**
A: Да, через `openclaw config set channels.telegram.accounts.<id>.allowFrom`.
Но это тема продвинутого модуля — не для этого workbook'а.

**Q: Сколько стоит использование API модели?**
A: Зависит от выбранной модели. GPT-5.4 через OpenAI ≈ $1–5/месяц
при обычном использовании. Бесплатные модели (minimax-m2.5-free)
— $0 но медленнее.

**Q: Могу ли я удалить одного агента?**
A: Да:
```bash
openclaw agents delete <имя-агента> --yes
openclaw channels remove --channel telegram --account <имя> --yes
rm -rf ~/.openclaw/workspace-<имя>
```
