# 🔑 Где взять OpenAI API-ключ для embedding-памяти

> ⏱ **Время:** 5-10 минут (включая регистрацию аккаунта если ещё нет)
> 💰 **Стоимость:** $5 минимальный депозит. Этого хватит на **годы**
> embedding-памяти для одного клиента (расход ≈$0.10-0.50 в месяц).

---

## 📍 Зачем нужен этот ключ

Установщик в шаге **R1.5** спрашивает: «Подключить embedding-память?».

Если соглашаешься — нужен **OpenAI API-ключ** для модели
`text-embedding-3-large`. Это не та же модель что отвечает в Telegram —
это специальная «индексирующая» модель, которая превращает твои тексты
в семантические векторы.

**Что даёт:** агент находит в своей памяти релевантный контекст по
смыслу (а не по точному совпадению слов). Через 2-3 месяца это
экономит $0.50-2/день на каждом активном клиенте.

---

## 🚀 Шаг 1 — Зарегистрироваться в OpenAI (если ещё нет)

1. Открой [https://platform.openai.com/signup](https://platform.openai.com/signup)
2. Зарегистрируйся: email + пароль (или через Google / Microsoft)
3. Подтверди email (придёт письмо)
4. Введи имя + дату рождения

`[SCREENSHOT: страница регистрации OpenAI]`

> 💡 **Уже есть ChatGPT-аккаунт?** Используй его — тот же email/пароль
> подходят для `platform.openai.com`. Это разные сервисы (ChatGPT для
> людей, platform для разработчиков), но один аккаунт.

---

## 🔑 Шаг 2 — Создать API-ключ

1. Зайди на [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Нажми **`+ Create new secret key`** (синяя кнопка справа сверху)
3. **Name** (необязательно): например, `openclaw-embedding`
4. **Project**: оставь `Default project`
5. **Permissions**: оставь `All`
6. Нажми `Create secret key`

`[SCREENSHOT: модалка Create secret key]`

OpenAI покажет твой ключ в виде:
```
sk-proj-AbCdEf1234567890...XyZ
```

⚠️ **ВАЖНО:** этот ключ показывается **только один раз**. Сразу скопируй
его и сохрани в надёжное место (например, в менеджер паролей).

Если потерял — придётся создать новый (старый можно отозвать там же).

---

## 💳 Шаг 3 — Положить $5 на счёт

OpenAI работает по **prepaid**-системе: сначала кладёшь деньги, потом
тратишь по факту использования. Без депозита embedding не сработает.

1. Зайди на [https://platform.openai.com/settings/organization/billing](https://platform.openai.com/settings/organization/billing)
2. Нажми **`Add payment details`**
3. Введи данные карты (поддерживаются Visa / Mastercard, в т.ч. российские
   через прокси-сервисы — см. ниже)
4. После добавления карты нажми **`Add to credit balance`**
5. Сумма: **минимум $5** (можешь $10 для запаса)
6. Подтверди оплату

`[SCREENSHOT: страница Billing с балансом]`

> 💡 **Сколько хватит $5?**
> Embedding-модель `text-embedding-3-large` стоит $0.13 за 1 миллион
> токенов. Активный клиент за месяц генерирует ~5 МБ памяти ≈
> 1.5 млн токенов. **Расход:** $0.20/месяц.
> $5 → **2 года** работы. $10 → 4 года.

### 💳 Карты РФ — что делать (100% не работают напрямую)

OpenAI **не принимает** карты выпущенные в России — это санкционное
ограничение, обходного пути «вставить нашу карту и оплатить» нет.
**Карта МИР, Visa/Mastercard выпущенные российскими банками — все
не пройдут.**

Что работает:

#### ✅ Вариант 1 — Зарубежная виртуальная карта через бот (быстро, ~10 минут)

Через Telegram-бот можно за минуты выпустить виртуальную **зарубежную
карту** (страна выпуска: Казахстан / Армения / другая) и пополнить её
с российской карты. Карта работает где угодно где принимают зарубежные
Visa / Mastercard — в т.ч. в OpenAI.

🔗 **Ссылка для регистрации:**
[https://t.me/WantToPayBot?start=w17851188--GUSNM](https://t.me/WantToPayBot?start=w17851188--GUSNM)

**Шаги:**
1. Открой ссылку в Telegram → нажми Start
2. В боте выбери выпуск виртуальной карты (зарубежной)
3. Пополни её рублями с любой российской карты (комиссия ~5-10%)
4. Получи реквизиты (номер карты + срок + CVV)
5. Используй эти реквизиты на странице
   [billing.openai.com](https://platform.openai.com/settings/organization/billing)
   → `Add payment details`
6. Положи $5-10 на счёт OpenAI

> 💡 **Сколько в рублях:** $5 на OpenAI ≈ 500-600₽ по курсу + ~50₽
> комиссия за пополнение карты + ~500₽ выпуск карты (один раз).
> Итого первый депозит — около **1000-1500₽**, дальше только пополнения
> (по 500-600₽ раз в год-два, embedding ест медленно).

#### ✅ Вариант 2 — Уже есть зарубежная карта

Если у тебя уже есть карта из:
- 🇰🇿 Казахстана
- 🇦🇲 Армении
- 🇬🇪 Грузии
- 🇹🇷 Турции
- 🇦🇪 ОАЭ
- любой страны Евросоюза / США / другой не-санкционной

— просто используй её на странице
[billing.openai.com](https://platform.openai.com/settings/organization/billing)
без посредников. Это самый дешёвый и быстрый вариант.

#### ✅ Вариант 3 — Попросить друга с зарубежной картой

Если у близкого человека есть зарубежная карта — пусть положит $5-10
себе на свой OpenAI-аккаунт, выдаст тебе ключ. Ты возвращаешь ему
рубли через СБП. Минус: ключ привязан к его аккаунту, он видит расходы.

#### ❌ Что НЕ работает

- Карты МИР, российские Visa/Mastercard
- Большинство криптокарт (OpenAI блокирует BIN'ы)
- Российские реквизиты в адресе (даже с зарубежной картой адрес лучше
  ставить казахстанский/армянский — иначе биллинг может зафейлить)

> ⚠️ **Не используй «общие» ключи** из Telegram-чатов или сайтов где
> «дам ключ за 100₽». Они могут перестать работать в любой момент,
> содержать payload-логирование, или попасть под массовую отзывку
> когда OpenAI заметит шеринг.

## ✅ Шаг 4 — Проверить что ключ работает

Простой `curl` для проверки:

```bash
curl https://api.openai.com/v1/embeddings \
  -H "Authorization: Bearer sk-proj-AbCdEf..." \
  -H "Content-Type: application/json" \
  -d '{"model":"text-embedding-3-large","input":"hello"}'
```

Если получаешь `{"object":"list","data":[{"embedding":[...]}]}` — **ключ работает**.

Если `{"error":{"message":"Incorrect API key..."}}` — ключ неверный.

Если `{"error":{"message":"You exceeded your current quota..."}}` — нет
денег на счёте.

Запусти установщик:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-agents-pack/main/scripts/install-agents.sh)
```

В шаге **R1.5** выбери:
- `1) Включить (рекомендуется)`

Установщик за 1-2 секунды проверит ключ через OpenAI API и пойдёт дальше.

## 🛡 Лучшие практики безопасности

### Не публикуй ключ в чатах

API-ключ = **деньги**. Если попадёт к злоумышленнику, он может:
- Сжечь твой $5 за минуту (пишет 100 000 запросов в OpenAI = $5)
- Использовать ключ для майнинга embedding'ов под чужие проекты

Что делать:
- ✅ Хранить в менеджере паролей (1Password / Bitwarden / KeePass)
- ❌ **Не** вставлять в Telegram-чаты, GitHub commits, скриншоты
- ✅ Если случайно расшарил — **сразу** зайди в
  [api-keys](https://platform.openai.com/api-keys), нажми `…` → `Revoke` →
  создай новый

### Установи лимит расходов

В [Settings → Limits](https://platform.openai.com/settings/organization/limits) можно
поставить:
- **Hard limit**: $20/месяц — больше не потратится
- **Email alerts**: на $5 / $10 / $15 — уведомление если расход растёт

Это защита от сбежавшего цикла или утечки ключа.

### Отдельный ключ только под embedding

Если у тебя один ключ работает и для chat-модели и для embedding —
это нормально, но:
- Если утечёт — сгорит **больше** (chat-модели в 100 раз дороже embedding)
- Сложнее понять «куда уходят деньги»

Я рекомендую:
- **Один ключ для chat-модели** (если используешь OpenAI как chat-модель)
- **Отдельный ключ только для embedding** (с лимитом $20/месяц)
- Назови их по-разному в `Name` поле при создании, чтобы потом не путаться

## ❓ FAQ

**Q: Я уже плачу за ChatGPT Plus ($20/месяц). Зачем ещё $5?**
A: ChatGPT Plus — это подписка для веб-интерфейса chat.openai.com.
**API — это отдельный сервис** с отдельным billing. Подписка
ChatGPT Plus не даёт API-кредитов. К сожалению.

**Q: Можно ли использовать другую модель для embedding (не OpenAI)?**
A: Сейчас установщик прибит к OpenAI `text-embedding-3-large`.
Поддержка других провайдеров (Anthropic, Google) — в планах wave 9
если будет запрос. Можно вручную править
`~/.openclaw/openclaw.json` → `agents.<id>.memorySearch.provider`,
но без поддержки на стороне OpenClaw оно не сработает.

**Q: Можно ли вообще без OpenAI? Бесплатные локальные embeddings?**
A: Технически да (sentence-transformers через Ollama), но:
- OpenClaw пока не поддерживает локальные embeddings
- Качество хуже чем у text-embedding-3-large
- Скорость — медленнее (CPU)
- Нужно RAM 4+ GB и стабильная нагрузка
Wave 9-10 если будет запрос.

**Q: Что если откажусь от embedding сейчас?**
A: Ничего страшного — агенты будут работать как раньше, читая
MEMORY.md целиком. Через несколько месяцев когда захочешь подключить —
запусти `--enable-embedding` или просто переустанови с флагом
`--refresh-templates` и в R1.5 выбери «Включить».

**Q: Сколько реально потратится в месяц?**
A: Зависит от активности:
- Малоактивный клиент (10 сессий/неделя) — $0.05/месяц
- Активный клиент (50 сессий/неделя) — $0.20/месяц
- Очень активный (200+ сессий/неделя) — $1-2/месяц

Embedding-расходы **не растут** с ростом качества ответов, в отличие
от chat-модели — это разовое индексирование MEMORY.md.
