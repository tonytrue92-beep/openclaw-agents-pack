Что у тебя в этом окне (DMG):
  📄 1. Установить OpenClaw.command   ← запусти ПЕРВЫМ
  📄 2. Установить AI-команду.command ← запусти ВТОРЫМ
  📄 README.txt                        ← этот файл

Алгоритм:
  1. Двойной клик на "1. Установить OpenClaw.command"
  2. Дождись пока бот ответит в Telegram (~5-10 минут)
  3. Двойной клик на "2. Установить AI-команду.command"
  4. Готово — все боты живые в Telegram

═══════════════════════════════════════════════════════════════
  ВНИМАНИЕ — первый запуск может показать предупреждение
═══════════════════════════════════════════════════════════════

При первом двойном клике macOS может показать:

  "Файл нельзя открыть, потому что разработчик не подтверждён"
  или
  "Apple cannot check it for malicious software"

Это НОРМАЛЬНО. Мы маленький проект, не платим Apple $99/год за
официальную подпись. Скрипты внутри безопасные — они просто тянут
команды установки с GitHub (можешь открыть .command в любом текстовом
редакторе и проверить).

КАК ОБОЙТИ — ОДИН РАЗ на каждый файл:

  1. Закрой окно с предупреждением (нажми "Отмена" или Cancel)

  2. ПРАВЫЙ КЛИК (или Control+клик) на файле .command в Finder

  3. В контекстном меню выбери "Открыть" (Open)

  4. Появится новое окно с кнопкой "Открыть" — нажми её

  5. Дальше работает как обычно. Повтори то же для второго файла.

Заранее подготовь:

  ✦ Курс-токен от @AITeamVIPBot
    → Открой Telegram → бот @AITeamVIPBot → /start
    → Введи email или телефон оплаты курса
    → Получишь STD-... (Standard) или VIP-... (VIP)

  ✦ Telegram-боты (3 или 6 в зависимости от тарифа)
    → Telegram → @BotFather → /newbot → создай ботов по одному
    → Сохрани каждый токен (длинная строка вида 123456:AABBcc...)

  ✦ Свой Telegram user ID
    → Telegram → @userinfobot → /start
    → Бот пришлёт твой ID (число типа 123456789)

  ✦ (Опционально) OpenAI API-ключ для embedding-памяти
    → Платформа OpenAI → settings/api-keys
    → Карты РФ напрямую не работают — для них есть путь через
      виртуальную карту (ссылку даст установщик в нужный момент)
