# Copywriting — conversion copy that sells

Скилл от `@staybased` даёт мне 6 проверенных direct-response фреймворков
для продающих текстов. Подключаю когда нужен текст **для конверсии**
(лендинг, объявление, описание продукта, CTA).

## Когда использую

- Лендинг-страницы и их секции (hero, features, social proof, pricing, CTA)
- Описания продуктов на маркетплейсах (Gumroad / ClawHub / и т.п.)
- Рекламные объявления (соц-сети, поиск, display)
- Улучшение существующего текста который плохо конвертит
- Profile-bio на биржах (Upwork и аналоги) — это тоже продающий текст
- CTA-кнопки и microcopy

## Когда НЕ использую

- Холодные email'ы (нужен отдельный фреймворк — cold-outreach)
- SEO-статьи в блог (цель — трафик, не моментальная конверсия)
- Технические README (ясность > убеждение)
- Коммерческие предложения (отдельный формат)

## 6 фреймворков (выбираю под задачу)

| Фреймворк | Когда брать |
|---|---|
| **PAS** (Problem → Agitate → Solve) | Лендинг под известную боль. Аудитория знает что проблема есть |
| **AIDA** (Attention → Interest → Desire → Action) | Холодная реклама, email subjects, общие продающие страницы |
| **FAB** (Features → Advantages → Benefits) | Описания продукта, feature-страницы, сравнительные таблицы |
| **BAB** (Before → After → Bridge) | Трансформация, кейсы, отзывы |
| **4P** (Promise → Picture → Proof → Push) | Длинные sales-pages |
| **Star-Story-Solution** | About-страницы, бренд-нарратив, личный бренд |

## Правила (из скилла)

1. Пишу для сканирующих — заголовки, буллиты, короткие абзацы
2. Одна идея = одно предложение
3. «Ты» > «Мы». Считаю «вы»-ы vs «мы»-и в своём тексте
4. Конкретика > общие слова. Цифры > прилагательные
5. Benefits > Features (всегда перевожу первое во второе)
6. CTA = глагол действия + результат («Начать автоматизировать» >
   «Нажать здесь»)
7. Первый черновик режу на 30%
8. Читаю вслух — если споткнулся, переписываю
